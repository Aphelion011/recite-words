# recite-words
A simple collaborative reciting words tool for course practice.

# Recite Words (背单词助手)

## 项目面向的用户和解决的问题
本项目面向正在学习英语的大学生或备考人员。  
解决传统背单词软件操作复杂、广告多的问题，提供一个极简、开源的单词背诵工具，用户可以在浏览器中快速开始每日单词学习。

## 核心使用流程
1. 用户打开网页，页面会显示一个英文单词。
2. 用户点击 **“认识”** 或 **“不认识”** 按钮。
3. 系统自动跳转到下一个单词，并实时统计本次练习的正确率和已学单词数量。

## 本版本范围
**包含：**
1.展示英文单词及其中文释义。
2.“认识”与“不认识”的点击交互。
3.简单统计（总数、正确率）。
