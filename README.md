# 📖 背单词助手 (Recite Words)

## 项目简介
本项目是一个极简的网页版背单词工具，支持多组词库切换（四级、六级、考研、雅思），适合大学生备考英语使用。

## 在线访问
直接访问 GitHub Pages 即可开始使用：  
🔗 [https://aphelion011.github.io/recite-words](https://aphelion011.github.io/recite-words)

## 本地运行方法
1. 克隆本仓库到本地：`git clone https://github.com/Aphelion011/recite-words.git`
2. 进入文件夹，双击 **`index.html`** 文件，在浏览器中打开即可。

## 核心功能
- ✅ 点击 **“认识”** / **“不认识”** 记录学习进度
- ✅ 实时统计：已学数量、正确数量、正确率
- ✅ 一键切换 **四级 / 六级 / 考研 / 雅思** 四组词库
- ✅ 点击 **“重来”** 重置当前词库进度
- ✅ 全部背完后显示庆祝完成界面
